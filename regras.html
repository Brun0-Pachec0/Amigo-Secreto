Instruções para o Amigo Secreto

  • Todos os participantes precisam estar juntos presencialmente;
  • Adicione o nome das pessoas que participaram no Amigo Secreto (um nome de cada vez);
  • Após adicionado todos os participantes, clique em 'Sortear'
  • Primeiro aparecerá o nome de um participante e logo após pressionar o botão 'Ver', quem esse participante tirou;
  • Para seguir basta clicar em 'Esconder' que o nome será apagado, podendo assim ser passado o celular ou a tela para a próxima pessoa ver quem ela tirou;

  • Não é possível adicionar dois ou mais nomes iguais (adicione sobrenome caso necessário);
  • O sorteio só é possível depois de adicionar pelo menos 2 pessoas;
  • Após todos os participantes verem quem respectivamente eles tiraram, o sorteio é finalizado;
  • Não é possível voltar para o participante anterior, e após a finalização do sorteio, todos os dados são apagados, não existindo nenhum tipo de histórico.
